<details open>
<summary><strong>🧭 About the Game</strong></summary>

<br>

**Wasteds** is a **third-person open-world survival RPG** set in a harsh, procedurally generated world where survival is never guaranteed.

Each new game creates a **unique land** filled with:
- Mysterious ruins 🏛
- Hostile magical creatures 👁
- Dangerous biomes and environmental threats 🌪

You begin alone.  
No map knowledge. No safety.  
Only your choices define the story.

</details>

---

<details>
<summary><strong>⚔ Core Features</strong></summary>

<br>

### 🌍 Procedural Open World
- Chunk-based terrain generation  
- Biomes, noise pipelines, erosion & height layers  

### 🧝 Character & Equipment
- Full character creation  
- Visible armor & weapons  
- NPCs share the same systems  

### 🔮 Risky Magic System
- Powerful spells with real consequences  
- High risk, high reward  

### 🧠 Hybrid Progression
- Attributes + skill trees  
- Inspired by Skyrim & World of Warcraft  

### 🎨 Visual Style
- Low-poly geometry  
- Hand-painted textures  
- Dark, atmospheric tone  

</details>

---

<details>
<summary><strong>🛠 Tech Overview</strong></summary>

<br>

- **Engine:** Godot 4.x  
- **Language:** GDScript  
- **World:** Chunk-based terrain generation  
- **Vegetation:** Rule-based procedural spawning  
- **Rendering:** Custom terrain shader  

Modular, system-driven architecture.

</details>
